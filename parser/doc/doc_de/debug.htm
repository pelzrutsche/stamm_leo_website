phpCMS besitzt einen Debug-Modus, der während der Entwicklung einer Website mit phpCMS hilfreich sein kann.

Der Aufruf einer Seite im Debug-Mode erfolgt durch das Anhängen des Schlüsselwortes "?debug=debug".
Der Aufruf sieht dann z.B. so aus:

http://www.yourdomain.com/some/document.htm?debug=debug

Der Debug-Modus ist Passwort-geschützt. Bitte vergessen Sie nach getaner Arbeit nicht, sich auszuloggen!!

Parameter Funktion debug lädt die Seite in der Debug-Ansicht content_file Content-Datei, ungeparst content_rep Content-Datei, mit ersetzten Tags template_file Template-Datei(en), ungeparst mit Zeilennummern template_rep Template-Datei(en), mit ersetzten Tags und Feldern tags alle verfügbaren Tags und deren Ersetzungen menu_file Menü-Datei(en), ungeparst menu_name Menü-Datei(en), nach NAME sortiert menu_class Menü-Datei(en), nach CLASS sortiert page Content-Datei, fertig geparst time Content-Datei, geparst mit Ausführungszeit (ohne PAX o. Scripte) vars alle über $DEFAULTS ansprechbaren Einstellungen von phpCMS dev Inhalt des $GLOBALS Arrays

Debug-mode muss im GUI aktiviert sein, um diese Parameter einzusetzen.
